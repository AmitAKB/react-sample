# AAD Ag-Grid Example (Frontend + Backend)

## Overview
- Frontend: Vite + React + TypeScript, uses MSAL config and Ag-Grid for data display.
- Backend: Simple Express server exposing /api/data.

## Quickstart
1. Frontend: cd frontend && npm install && cp .env.example .env && (edit .env) && npm run dev
2. Backend: cd backend && npm install && npm start
